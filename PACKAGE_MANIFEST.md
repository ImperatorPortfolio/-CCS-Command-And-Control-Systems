# Package Manifest

## Scope

New standalone Space Engineers mod implementing the CommandOS weapon-management vertical slice and its public integration SDK.

## Status

**Build-candidate**

## Exact replacement scope

Install or replace the complete local directory:

`%APPDATA%\SpaceEngineers\Mods\CommandOS`

No existing weapon-pack source file is replaced by this package. The files in `SDK` are complete source files intended for a subsequent weapon-pack integration package.
