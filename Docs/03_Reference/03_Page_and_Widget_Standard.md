# Page and Widget Standard

**Product:** CommandShellOS for Space Engineers  
**Document type:** UI reference  
**Package version:** 0.1.0  
**Status:** Build-candidate  
**Prepared:** 2026-06-29

---

Widgets: status tile, numeric readout with unit/quality, capacity/endurance bar, trend graph, synoptic, entity/contact table, tactical/navigation/compartment map, procedure checklist, alarm list, timeline, camera, selector, bounded slider, normal command, guarded command, multi-party authorisation, and context inspector.

Every widget supports focus, hover, disabled reason, pending/result state, pointer hit bounds, and accessible critical indication. A page definition contains ID/version/category/title claim/station types/layout/widgets/subscriptions/commands/context actions/alarm links/help/theme. Widgets never search blocks or issue direct game actions.
