# Alert Condition Profile Matrix

**Product:** CommandShellOS for Space Engineers  
**Document type:** Reference  
**Package version:** 0.1.0  
**Status:** Build-candidate  
**Prepared:** 2026-06-29

---

| Condition | Purpose | Baseline posture |
| --- | --- | --- |
| White | Maintenance/refit | Weapons safe, maintenance access, isolation permitted |
| Green | Normal operations | Standard power, passive watch, normal access |
| Blue | Flight operations | Hangars controlled, traffic/rescue ready, flight zones restricted |
| Yellow | Elevated readiness | Reserves prepared, sensors intensified, crew alerted |
| Red | General quarters | Combat plant, weapons by ROE, compartments secured, damage teams ready |
| Black | Catastrophic emergency | Survival, shelters, evacuation, distress, and scuttle availability |

Each vessel publishes versioned transition procedures rather than hardcoding page actions.
