## Taking a Station

1. Confirm the vessel and station shown in the header.
2. Authenticate using the configured server identity.
3. Accept the assigned watch role.
4. Take control where the station requires exclusive ownership.
5. Review handover, alarms, procedures, lockouts, and pending commands.

## Pointer Use

Hover/focus identifies an interactive target. Primary click selects or activates. Guarded controls open a consequence panel first. Long operations show pending state and a separate cancel/abort action where supported.

## State Quality

Valid is current authoritative state. Estimated is calculated. Stale exceeded allowed age. Conflicted means sources disagree. Unavailable means no trustworthy state. Never infer success from a screen animation.

## Alarms

Acknowledge means seen, not repaired. Assign, investigate, execute the appropriate procedure, verify clearance, and close with evidence.

## Ending a Watch

Resolve or hand over active work, state temporary limitations, review alarms with the relief, transfer control positively, and lock/end the session.
